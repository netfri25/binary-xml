# Binary XML encode/decode

<img src='assets/showcase.png'>

## Getting Started

#### Dependencies
 - fasm
 - make

#### Build
```shell
make
```

#### Clean
```shell
make clean
```

## Usage

#### Encoding
```shell
build/encode < [file to encode] > [encoded file]
```

#### Decoding
```shell
build/decode < [file to decode] > [decoded file]
```
