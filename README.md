# Binary XML encode/decode

<img src='assets/showcase.png'>

## Getting Started

#### Dependencies
 - gcc
 - make

#### Build
```shell
make
```

#### Clean
```shell
make clean
```

## Usage

#### Encoding
```shell
cat <file-to-encode> | ./encode > <encoded-file>
```

#### Decoding
```shell
cat <file-to-decode> | ./decode > <decoded-file>
```
